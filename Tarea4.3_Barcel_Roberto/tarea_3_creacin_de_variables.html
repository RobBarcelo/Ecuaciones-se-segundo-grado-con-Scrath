Tarea 3: Creación de variables

Creación de variables

A continuación, vamos a  aprender sobre el uso de los CONTROLADORES y, en especial, sobre el CONDICIONAL "SI".

También indagaremos sobre el funcionamiento de:

- Sensor "PREGUNTAR".
- Creación de VARIABLES.
- Funcionamiento de OPERADORES.

El siguiente vídeo os servirá para resolver posibles dudas:

Tarea 3: Crea:

- Las variables a,b y c que tiene una ecuación de segundo grado.
- La variable d (discriminante) utilizando un operador.
- Los sensores "PREGUNTAR", "¿Cuánto vale...?" y "ESPERAR" para solicitar las variables a, b y c.
